🏠 Analyse et Prédiction des Prêts Immobiliers avec Python

Ce projet consiste à analyser les demandes de prêts immobiliers en utilisant des données sur les emprunteurs. Le modèle de Machine Learning prédit la probabilité d'acceptation ou de refus des prêts, tout en fournissant des alertes sur les montants élevés pour le directeur d'agence. Les outils utilisés incluent pandas, scikit-learn et matplotlib.

🎬 Analyse de Sentiments sur des Critiques de Films avec NLP

Ce projet NLP applique des techniques classiques (Word2Vec) et modernes (Transformers) pour analyser des critiques de films et séries. L'objectif est de prédire si une critique est positive ou négative. Le pipeline comprend du nettoyage de texte, de la vectorisation, et l'utilisation du modèle de classification Hugging Face. Outils : pandas, gensim, scikit-learn, transformers.

📈 Régression Linéaire avec Descente de Gradient

Ce projet implémente une régression linéaire à partir de zéro, en utilisant la méthode de descente de gradient pour ajuster les paramètres. Le modèle prédit une variable en fonction d'une caractéristique, tout en optimisant l'erreur de prédiction. Ce projet est écrit en Python avec l'utilisation des bibliothèques Numpy et Matplotlib.
